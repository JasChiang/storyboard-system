# 場景視頻自動載入修正說明

## 問題描述

原本的實作只是單純嵌入空的 Twick Studio 編輯器，沒有載入已生成的場景視頻。

## 修正內容

### 1. 新增時間軸資料轉換函數

檔案：`lib/utils/twick-converter.ts`

新增函數：`convertToTwickTimelineData()`

功能：
- 將場景陣列轉換為 Twick Studio 的時間軸資料格式
- 自動識別視頻或圖片素材
- 建立視頻軌道和字幕軌道
- 套用 AI 建議的時長調整

資料結構：
```typescript
{
  tracks: [
    {
      type: 'video',
      id: 'main-track',
      name: '主軌道',
      elements: [
        {
          id: 'scene-xxx',
          trackId: 'main-track',
          name: '場景 1',
          type: 'video',  // 或 'image'
          s: 0,           // 開始時間
          e: 5,           // 結束時間
          props: {
            src: 'video-url'  // 場景視頻 URL
          }
        }
      ]
    },
    {
      type: 'caption',
      id: 'caption-track',
      name: '字幕軌道',
      elements: [...]
    }
  ],
  version: 1
}
```

### 2. 更新組件初始化

檔案：`components/export/TwickStudioWrapper.tsx`

修改：
- 使用 `convertToTwickTimelineData()` 轉換場景資料
- 將轉換後的資料傳給 `TimelineProvider` 的 `initialData`
- 移除空的 `INITIAL_TIMELINE_DATA`

修改前：
```tsx
<TimelineProvider initialData={INITIAL_TIMELINE_DATA} ...>
```

修改後：
```tsx
<TimelineProvider
  initialData={convertToTwickTimelineData(storyboard.scenes, aiSuggestion)}
  ...>
```

## 現在的行為

### 自動載入場景

當進入時間軸編輯模式時：
1. 系統自動讀取所有已生成的場景視頻
2. 轉換為 Twick Studio 時間軸格式
3. 載入到編輯器的主軌道中
4. 如果有字幕，同時載入到字幕軌道

### 支援的素材類型

- 視頻（generatedVideo）：直接載入為視頻元素
- 圖片（generatedImage）：載入為圖片元素，可設定顯示時長

### AI 建議整合

如果有執行 AI 分析：
- 自動套用建議的場景時長（inPoint 到 outPoint）
- 字幕自動加入對應軌道
- 準備好轉場和效果資料（等待套用）

## 使用方式

1. 建立專案並生成場景視頻

2. 進入導出頁面

3. 選擇「時間軸可視化編輯」模式

4. 編輯器會自動載入：
   - 所有場景視頻按順序排列在主軌道
   - 場景字幕顯示在字幕軌道
   - 可直接在時間軸上編輯

5. 可選執行 AI 分析並套用建議

6. 編輯完成後導出視頻

## 技術細節

### 時間軸元素屬性

每個場景元素包含：
- `id`：場景唯一識別碼
- `trackId`：所屬軌道 ID
- `name`：場景名稱（場景 N）
- `type`：元素類型（video 或 image）
- `s`：開始時間（秒）
- `e`：結束時間（秒）
- `props.src`：視頻或圖片 URL

### 字幕元素屬性

每個字幕元素包含：
- `type`：'caption'
- `s` 和 `e`：對應場景的時間範圍
- `props.text`：字幕文字
- `props.fill`：文字顏色
- `props.bgColor`：背景顏色

## 與其他功能的整合

### FFmpeg 渲染

編輯後的時間軸可以：
- 導出更新的場景順序
- 導出調整後的時長
- 傳送給 FFmpeg 進行最終渲染

### AI 建議

AI 分析結果會：
- 自動調整場景時長
- 提供字幕文字
- 準備轉場和效果設定

## 已知限制

1. 轉場和效果需要額外套用
   - 目前只載入基本場景和字幕
   - 轉場和視覺效果需要進一步整合

2. 僅載入有素材的場景
   - 沒有生成視頻或圖片的場景會被跳過

3. 圖片場景
   - 圖片會作為靜態畫面顯示設定的時長
